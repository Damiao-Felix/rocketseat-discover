# Audio

O audio e seus atributos

* src
* controls
* se não funcionar?
  * fallback content
* source
  * src
  * type

```html
<audio controls autoplay muted>
  <source src="./assets-example/viper.mp3"
    type="audio/mp3">
  <source src="./assets-example/viper.ogg"
    type="audio/pgg">
    <p>Seu navegador não suporta o audio</p>
</audio>
```
