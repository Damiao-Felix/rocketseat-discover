# Tipos de dados

Conforme o ECMAScript standard temos 9 tipos de dados

* Data Types
    * Primitive / Primitive value
    * Structural
    * Strutural Primitive

## Primitivos

* String
* Number
* Boolean
* Undefined
* Symbol
* bigInt

## Estruturais

* Object
    * Array
    * Map
    * Set
    * Date
    * ...
* Function

## Primitivo Estrutural / Strutural Root Primitive

* Null
